## 🧠 Ключові концепції ООП, які я застосовую:
* **Інкапсуляція:** Обмеження прямого доступу до методів та атрибутів класу (використання приватних `__` та захищених `_` модифікаторів, а також `@property`).
* **Успадкування:** Створення ієрархій класів для уникнення дублювання коду (DRY - Don't Repeat Yourself).
* **Поліморфізм:** Використання єдиного інтерфейсу для обробки об'єктів різних типів (перевизначення методів).
* **Абстракція:** Створення абстрактних класів та методів (модуль `abc`) для побудови гнучких архітектурних шаблонів.
